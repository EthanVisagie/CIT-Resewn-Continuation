## CIT Resewn Continuation 1.2.2-fork.5+26.1.2

Beta compatibility release for Minecraft 26.1, 26.1.1, and 26.1.2 on Fabric.

### Fixed

- Ported the 26.1.x line to Fabric Loader 0.19.2 and Fabric API 0.149.0+26.1.2.
- Restored Barely Default elytra CIT loading on 26.1.x.
- Fixed damaged/worn elytra variant priority so low-durability textures resolve correctly.
- Restored material-prefix fallback for item model names such as `Diamond Claymore`, `Golden Cutlass`, and similar variants.
- Kept the defaults module bundled inside the main jar.

### Install

Install only the main jar:

- `citresewn-continuation-1.2.2-fork.5+26.1.2.jar`

Do not install the standalone defaults jar; it is bundled inside the main jar.
